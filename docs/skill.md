# スキル一覧

スキルIDは skill.dat.js に定義されている

例：
ジュデックスのスキルIDは476
精錬値9以上のときBaseLvが2上がる度にジュデックスの威力が上昇する効果は 20000009005476
```
2       # BaseLv n 毎に発動する
00000   # 
09      # 精錬値 n 以上で発動する
0       # 精錬値 n 毎に発動する
05      # スキルの威力増加
476     # スキルID
```

例
クレッシブボルトのスキルIDは1051
この場合はスキルの威力増加コードと加算する
```
2       # BaseLv n 毎に発動する
00000   # 
09      # 精錬値 n 以上で発動する
0       # 精錬値 n 毎に発動する
06      # スキルの威力増加 (05) + スキルIDの1桁 (1)
051     # スキルIDの3桁
```
